## Academic Project

This project was developed as part of the **Machine Learning course** at  
**Université Internationale de Rabat (UIR)** during my **Licence program**.

The objective of this project was to apply classical machine learning techniques
to a real dataset and build a model capable of predicting diabetes based on
medical diagnostic measurements.

### Project Tasks
- Data preprocessing and cleaning
- Handling missing values
- Feature scaling
- Training classification models
- Model evaluation using Accuracy, Confusion Matrix, and ROC Curve

### Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
